How it works

Three steps. No magic, no overselling.

Here's exactly what it costs and exactly what you get.

1

Install the extension

Add OpenJam from the Chrome Web Store — one click, auto-updates. Or load the release zip unpacked.

Cost: it's a Chrome extension · Chromium browsers, Chrome 118+
2

Hit record & talk

OpenJam attaches via the same DevTools debugger protocol Chrome ships with — you'll see the "being debugged" banner. Reproduce the bug and narrate what you're doing.

Chrome shows a "being debugged" banner — that's the capture working
3

Get one HTML file

Stop & download a self-contained report — session replay, full timeline, and your narration. Open it offline, send it however you already send files.

Output: 1 .html file · opens offline, no viewer needed
